1. Herunterladen des Plugins
============================

Laden Sie das Plugin von unserer Plugin-Seite herunter.

2. Installation des Plugins
===========================

Melden Sie sich in der Administrations-Oberfläche Ihrer Joomla!-Installation an.

Um das Plugin zu installieren, öffnen Sie den Reiter "Upload Package File" in der Erweiterungsübersicht, die Sie im Menü unter "Extensions > Manage" finden.
Klicken Sie hier auf "Datei auswählen" und wählen Sie die Datei, die Sie zuvor herunter geladen haben.
Klicken Sie nun auf "Upload & Install".

3. Einrichten des Plugins
=========================

Besuchen Sie nun die Einstellungsseite des Newsletter2Go-Plugins, die sie im Menü unter "Components > Newsletter2Go" finden.

3.1 Connect to Newsletter2Go
----------------------------

Tragen Sie Ihren API-Schlüssel in das Textfeld ein.
Sie finden den Schlüssel im Reiter "Plug-ins" in den Einstellungen Ihres Newsletter2Go-Benutzerkontos.
Um dorthin zu gelangen, können Sie den bereitgestellten Link "Where do I find my API key" verwenden.

3.2 Double Opt In Code
----------------------

Tragen Sie den Double Opt In Code ihres Anmeldeformulars in das Textfeld ein und klicken Sie auf "Save".
Sie finden den Code im Reiter "Anmeldeformular" in den Einstellungen Ihres Newsletter2Go-Benutzerkontos.
Um dorthin zu gelangen, können Sie den bereitgestellten Link "Where do I find my DOI code" verwenden.

3.3 Configure subscription form (optional)
------------------------------------------

Zuletzt können Sie die Einstellungen Ihres Anmeldeformulars bearbeiten.
Hier können Sie z.B. auswählen, welche Felder angezeigt werden sollen und in welcher Reihenfolge.
Sie können den Feldern außerdem eigene Bezeichnungen zuweisen und festlegen, ob diese ausgefüllt werden müssen, oder nicht.
